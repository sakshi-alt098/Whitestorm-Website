# Scope: Global CSS Conversion

## Objectives
Convert the WhiteStormm theme from dark to light mode by modifying the root CSS variables and global selectors in `src/index.css`.
The aesthetic must be minimalist, luxurious, and clean (resembling an Apple product reveal).

## Tasks
- Update `:root` variables:
  - `--bg-color`: light background value (e.g. `#fafafa` or `#f8fafc`).
  - `--text-color`: dark text color value (e.g. `#0f172a` or `#1e293b`).
  - Update any metallic variables (`--metal-*`) to lighter, clean aluminum/titanium gradients.
- Modify global styling:
  - Update `body` background settings to handle the light theme.
  - Review scroll effects, default button styles, and headings (`h1, h2, h3` gradients/colors).
- Ensure that the initial light theme compiles without errors.
